                            < Folder Defenderとは >
  Folder Defenderはお使いのPC上にあるフォルダ/ファイルに疑似的にパスワードを
設定し非表示にするソフトウェアです。大切なデータや他人に見せたくないデータを隠す
ことができます。

                             < 使用上の注意事項 >
  フォルダ/ファイルのパスワード機能は本ソフトウェアで対象のデータをアンロックする際に
制限をかけるものです。本ソフトウェアを使用せずにアンロック(可視化)する場合はパスワード
は効果がありません。
本ソフトウェアは対象のフォルダ/ファイルを非表示にするものです。
セキュリティーソフトではありません。
※隠しファイルの表示が有効になっているPCでは非表示になりません!

                                  < 動作環境 >
・Windows 7 / Windows 8 / Windows 8.1 / Windows 10
※本ソフトウェアを実行するためにはJavaがインストールされている必要があります!
